# 真机录屏脚本（Guideline 1.2）

录屏要求：
- 使用真实 iPhone / iPad（不要模拟器）。
- 全程连续录制，不剪关键步骤。
- 建议每步停留 2-3 秒，便于审核员识别。

## 演示步骤

1. 启动 App（新安装或新账号）。
2. 进入任一 UGC 页面（如 Campus / Forum / Course / Messages）。
3. 展示 EULA/条款弹窗，并点击“接受并继续”。
4. 打开一条 UGC 内容（帖子/评论/消息）的操作菜单，点击“举报”。
5. 选择举报原因，展示“已举报”反馈。
6. 在同一内容或私信会话中打开操作菜单，点击“屏蔽用户”。
7. 确认屏蔽，展示成功提示。
8. 立即返回列表/会话，展示该用户内容已消失。
9. 下拉刷新或重新进入页面，展示该用户内容仍被过滤。
10. 进入 Profile > Help & Support，展示联系邮箱。
