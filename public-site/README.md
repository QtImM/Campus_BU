# HKCampus Public Site

This folder is a minimal static site intended for Vercel deployment.

## Deploy on Vercel

1. Import this repository into Vercel.
2. Set the project's **Root Directory** to `public-site`.
3. Deploy.

## Public URLs

After deployment, use these production paths:

- `/privacy` for the Privacy Policy URL in App Store Connect
- `/support` for the Support URL

The raw HTML files also remain available as:

- `/privacy-policy.html`
- `/support.html`
